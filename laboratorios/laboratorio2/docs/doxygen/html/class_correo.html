bool Correo::verificar ( )

Método para verificar un correo ingresado.

Se solicita un correo por medio de la terminal y se compara con expresion.

Devuelve
true Si el correo coincide con patrón dado en expresion.
false Si el correo no coincide con patrón dado en expresion.
