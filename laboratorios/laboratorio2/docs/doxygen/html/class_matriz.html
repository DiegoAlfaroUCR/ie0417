template<class T>
class Matriz< T >

Clase base de matrices.

En esta clase se contienen las entradas del objeto matriz en contenido. Se emplean métodos para asignar los valores de la matriz y para pedir la operacion a realizar en las matrices.

Parámetros de plantilla
Ttipo de variable usada (int, float o complex<float>).
