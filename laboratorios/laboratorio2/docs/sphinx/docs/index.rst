Simulador MIPS documentation
============================

Este proyecto consiste en un simulador básico de un subconjunto de instrucciones del conjunto de instrucciones MIPS.
El objetivo principal es proporcionar una herramienta sencilla para comprender el funcionamiento de algunas instrucciones
fundamentales de la arquitectura MIPS, incluyendo operaciones aritméticas, lógicas, de carga y almacenamiento,
saltos y desplazamientos.

Este simulador lee un archivo de texto que contiene código ensamblador MIPS y emula la ejecución de dichas instrucciones
en un conjunto limitado de registros y una memoria simulada.
